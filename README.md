# Breast Cancer Dataset

## Overview

This dataset contains diagnostic measurements of breast cancer cell nuclei and is commonly used for machine learning classification tasks.

## Dataset Information

* Total Records: 569
* Total Features: 30
* Target Variable: Diagnosis

### Diagnosis Labels

* B = Benign
* M = Malignant

## Feature Categories

The dataset includes measurements related to:

* Radius
* Texture
* Perimeter
* Area
* Smoothness
* Compactness
* Concavity
* Symmetry
* Fractal Dimension

## Applications

This dataset can be used for:

* Machine Learning Classification
* Data Analysis
* Feature Engineering
* Model Evaluation
* Healthcare Analytics
